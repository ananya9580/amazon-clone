# Amazon Clone

A simple Amazon clone built using **HTML & CSS**.  
This project replicates the basic layout and design of Amazon's homepage.

## Features
- Navigation bar with logo and links
- Product sections with images
- Footer with useful links

## Technologies Used
- HTML
- CSS

## How to View
Open `index.html` in any browser to see the project.
